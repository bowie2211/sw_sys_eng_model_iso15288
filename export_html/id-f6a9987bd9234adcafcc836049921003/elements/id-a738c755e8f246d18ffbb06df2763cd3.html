Disposal Process ()
Purpose As stated in ISO/IEC/IEEE 15288, The purpose of the Disposal process is to end the existence of a system element or system for a specified intended use, appropriately handle replaced or retired elements, appropriately handle any waste products, and to properly attend to identified critical disposal needs.
