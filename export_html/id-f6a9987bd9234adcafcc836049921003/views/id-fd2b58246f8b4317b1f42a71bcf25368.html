Configuration Management Process ()
Purpose As stated in ISO/IEC/IEEE 15288, The purpose of the Configuration Management (CM) process is to manage system and system element configurations over their life cycle. Actvities Prepare for configuration management Perform configuration identification. Perform configuration change management. Perform configuration status accounting. Perform configuration verification and audit.
