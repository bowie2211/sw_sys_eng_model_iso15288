Stakeholder Needs and Requirements Definition ()
Purpose As stated in ISO/IEC/IEEE 15288, The purpose of the Stakeholder Needs and Requirements Definition process is to define the stakeholder needs and requirements for a system that can provide the capabilities needed by users and other stakeholders in a defined environment.
