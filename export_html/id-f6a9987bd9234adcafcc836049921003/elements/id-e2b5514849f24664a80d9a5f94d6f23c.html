Design Definition Process ()
Purpose As stated in ISO/IEC/IEEE 15288, The purpose of the Design Definition process is to provide sufficient detailed data and information about the system and its elements to realize the solution in accordance with the system requirements and architecture. Activities Prepare for design definition • Create the system design • Evaluate the system design • Manage results of design definition
