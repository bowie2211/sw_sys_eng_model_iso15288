Decision Management Process ()
Purpose As defined by ISO/IEC/IEEE 15288 The purpose of the Decision Management process is to provide a structured, analytical framework for objectively identifying, characterizing and evaluating a set of alternatives for a decision at any point in the life cycle and select the most beneficial course of action.
